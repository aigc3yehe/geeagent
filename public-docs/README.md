# GeeAgent Public Docs

This folder contains the public-facing technical documentation draft for GeeAgent.
The audience is GeeAgent's open-source collaborators: people who need a clear
reference for public technical concepts, shipped behavior, and contribution
boundaries.

The site is intentionally simple:

- `index.html` is a no-dependency static Markdown viewer.
- `docs/en/` contains English Markdown.
- `docs/zh-CN/` contains Simplified Chinese Markdown.
- `docs/ja/` contains Japanese Markdown.

The visual style should remain plain black and white. This is a contributor
reference surface, not a marketing site.

When product-visible system behavior changes, check this folder and update the
relevant Markdown in all supported languages.
