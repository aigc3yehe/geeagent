# BTC Pulse Gear Package

`btc.price` is a Home widget gear for BTC/USD spot price display.

Current implementation status:

- Manifest and package metadata live in this folder.
- Widget Swift view is currently compiled from the host Home widget layer.
- The target package standard is to move widget-specific source, assets, setup
  files, and private dependencies under this folder.

This gear currently has no external dependency install plan.
